# 贡献说明

欢迎为这个仓库提交改进建议和代码。

## 提交前建议

- 保持目录结构清晰
- 不上传编译产物、缓存文件和临时文件
- 为新增功能补充最基本的说明
- 修改 README，保证别人能看懂项目怎么用

## 提交信息建议

建议使用简短明确的提交说明，例如：

```text
add project skeleton
fix README typo
update usage docs
```

## 代码整理建议

- 文件命名尽量统一
- 避免把多个不相关功能混在同一次提交中
- 删除无用文件后及时更新说明文档

## Issue 和反馈

如果后续仓库公开，建议优先通过 Issue 反馈：

- Bug 描述
- 复现步骤
- 运行环境
- 截图或日志

## 许可证

当前仓库尚未固定许可证。正式公开项目之前，建议补充 `LICENSE` 文件。

# GitHub 发布步骤

这份文档用于把源码和 Windows 便携版一起公开到 GitHub。

## 一、推送源码仓库

在你自己的 `cmd` 终端里进入仓库目录：

```bat
cd /d C:\Users\Administrator\programs
git status
git add -A
git commit -m "publish CircuitJS desktop source"
git push origin main
```

## 二、上传便携版压缩包

不要把 `electron-v9.3.2-win32-x64` 整个目录直接提交到仓库代码里。

原因：

- `electron.exe` 单文件超过 GitHub 普通仓库 100MB 限制
- 便携版更适合通过 GitHub Releases 发布

## 三、在 GitHub 网页创建 Release

1. 打开你的仓库主页
2. 点击右侧或顶部的 `Releases`
3. 点击 `Draft a new release`
4. Tag 可以写：`v0.1.0`
5. Title 可以写：`CircuitJS Desktop Portable v0.1.0`
6. 在描述里简单写：

```text
首次公开发布的 Windows 64 位离线便携版。
基于 CircuitJS1 整理。
```

7. 把本地生成好的 zip 压缩包拖进去作为附件
8. 点击 `Publish release`

## 四、建议上传的附件名

建议使用：

```text
circuitjs-desktop-windows-x64-portable.zip
```

## 五、发布后用户怎么使用

用户下载 zip 后：

1. 解压到本地任意目录
2. 双击 `electron.exe`
3. 即可离线运行

## 六、说明

如果后续你换了程序名或图标，Release 文件名也可以再改得更正式。

# programs

这是一个用于整理、托管和发布开源项目的代码仓库。

## 仓库用途

- 存放后续公开的项目源码
- 记录项目说明、使用方法和更新内容
- 作为个人开源项目的统一入口

## 当前状态

仓库已经清理旧的单片机示例文件，后续将用于新的开源项目。

## 建议的目录组织

你后续可以按下面的方式整理内容：

```text
programs/
  project-a/
  project-b/
  docs/
  README.md
  .gitignore
  CONTRIBUTING.md
```

如果以后只想维护一个项目，也可以直接把项目文件放在仓库根目录。

## 开源前建议准备

- 写清楚项目是做什么的
- 补充安装和运行说明
- 加上截图、演示或使用示例
- 选择合适的开源许可证
- 保持提交信息简洁清楚

## 后续可补充内容

- `LICENSE`
- 项目截图
- 安装步骤
- 常见问题
- 更新日志

## 维护说明

本仓库当前作为开源项目准备仓库使用，后续内容会持续整理。
